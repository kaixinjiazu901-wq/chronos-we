## 📋 生成体验版二维码步骤

### 第一步：下载并安装微信开发者工具

1. **访问下载页面**
   - 官方地址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

2. **选择对应系统版本**
   - Windows：下载 `.exe` 安装包
   - macOS：下载 `.dmg` 安装包
   - Linux：下载对应的安装包

3. **安装并打开**
   - 双击安装包，按照提示完成安装
   - 打开微信开发者工具

---

### 第二步：登录微信开发者工具

1. **扫码登录**
   - 使用微信扫描屏幕上的二维码
   - 确认登录

2. **选择项目类型**
   - 选择"小程序"
   - 点击"导入项目"

---

### 第三步：导入 Chronos 项目

**重要**：需要将 `dist` 文件夹从服务器下载到本地电脑

#### 方式 1：通过压缩包下载（推荐）

1. **在服务器上压缩 dist 文件夹**
```bash
cd /workspace/projects
zip -r chronos-dist.zip dist/
```

2. **下载压缩包**
   - 使用 FTP、SFTP 或其他方式下载 `chronos-dist.zip`
   - 或通过 scp 命令：
```bash
# 在本地电脑执行（替换服务器地址）
scp username@your-server:/workspace/projects/chronos-dist.zip ./
```

3. **解压到本地**
   - 解压 `chronos-dist.zip`
   - 记住解压后的 `dist` 文件夹路径

#### 方式 2：通过 Git 克隆（如果项目在 Git 仓库）

```bash
# 克隆项目到本地
git clone your-repo-url
cd your-repo
pnpm build:weapp
```

---

### 第四步：导入项目到微信开发者工具

1. **打开微信开发者工具**
2. **点击"+" 按钮**
3. **填写项目信息**：
   - **项目目录**：选择本地电脑上的 `dist` 文件夹
   - **AppID**：使用默认的 `touristappid`（测试用）
   - **项目名称**：Chronos
   - **开发模式**：小程序

4. **点击"导入"按钮**

---

### 第五步：检查项目配置

1. **确认项目导入成功**
   - 左侧显示页面列表：index、search-time、city-select、time-travel、city-showcase
   - 右侧显示预览界面

2. **测试基本功能**
   - 点击"编译"按钮
   - 在模拟器中测试功能是否正常

---

### 第六步：上传代码到微信平台

1. **点击"上传"按钮**
   - 在微信开发者工具顶部工具栏
   - 图标：☁️（云朵图标）

2. **填写版本信息**：
   - **项目版本号**：`1.0.0`
   - **项目备注**：Chronos 体验版 - 精神时区切换工具

3. **点击"上传"按钮**
   - 等待上传完成（通常几秒钟）

---

### 第七步：生成体验版二维码

1. **登录微信公众平台**
   - 访问：https://mp.weixin.qq.com/
   - 使用微信扫码登录

2. **进入版本管理**
   - 左侧菜单：管理 → 版本管理
   - 点击"体验版本"标签

3. **选择体验版本**
   - 找到刚才上传的版本（版本号：1.0.0）
   - 点击右侧的"▼"下拉菜单
   - 选择"体验版二维码"

4. **生成并保存二维码**
   - 系统生成体验版二维码
   - 右键保存二维码图片

---

### 第八步：分享体验版

1. **设置体验成员（可选）**
   - 在微信公众平台左侧菜单：管理 → 成员管理
   - 添加体验成员（最多 20 人）
   - 输入体验成员的微信号

2. **分享二维码**
   - 将保存的二维码图片发送给体验用户
   - 用户使用微信扫码即可体验

---

## 🔧 常见问题

### Q1: 上传失败怎么办？

**A**: 检查以下几点：
- 网络连接是否正常
- AppID 是否正确
- 项目目录是否正确
- 微信开发者工具是否已登录

### Q2: 体验版二维码扫不出来？

**A**: 可能的原因：
- 二维码图片不清晰
- 微信版本过低（建议升级到最新版）
- 体验成员未添加（如果设置了限制）

### Q3: 如何更新体验版？

**A**: 修改代码后重新上传：
1. 修改代码并重新构建：`pnpm build:weapp`
2. 下载新的 dist 文件夹
3. 在微信开发者工具中重新上传
4. 生成新的体验版二维码

### Q4: 体验版数据会丢失吗？

**A**: 不会。体验版使用本地存储，数据保存在用户手机上，重新上传代码不会影响用户数据。

### Q5: 体验版能转为正式版吗？

**A**: 可以。步骤：
1. 注册小程序账号获取真实 AppID
2. 更新 project.config.json 中的 appid
3. 重新构建和上传
4. 提交审核
5. 审核通过后发布正式版
