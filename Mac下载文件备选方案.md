# Chronos 文件下载 - Mac 用户备选方案

## 🎯 方案一：使用 SFTP 客户端（推荐）

### 使用 Cyberduck（免费，Mac 原生体验）

1. **下载 Cyberduck**
   - 访问：https://cyberduck.io/
   - 下载 macOS 版本并安装

2. **连接到服务器**
   - 打开 Cyberduck
   - 点击"打开连接"按钮（或按 `Cmd + N`）
   - 选择"SFTP (SSH File Transfer Protocol)"
   - 填写连接信息：
     - 服务器：`9.97.62.166`
     - 用户名：`root`（或其他用户名）
     - 密码：你需要向我获取服务器的登录密码

3. **下载文件**
   - 连接成功后，导航到 `/workspace/projects/`
   - 找到 `dist` 文件夹
   - 右键点击 → "下载"
   - 选择保存位置（桌面）

### 使用 FileZilla（免费，跨平台）

1. **下载 FileZilla**
   - 访问：https://filezilla-project.org/
   - 下载 macOS 版本并安装

2. **连接到服务器**
   - 打开 FileZilla
   - 在顶部填写连接信息：
     - 主机：`9.97.62.166`
     - 用户名：`root`
     - 密码：需要我提供
     - 端口：22
   - 点击"快速连接"

3. **下载文件**
   - 在左侧（本地站点）选择桌面
   - 在右侧（远程站点）导航到 `/workspace/projects/dist`
   - 右键点击文件 → "下载"

## 🎯 方案二：使用终端 SCP 命令（如果你有服务器密码）

如果你有服务器的登录密码，可以使用 SSH 命令下载：

```bash
# 打开终端
scp root@9.97.62.166:/workspace/projects/dist ~/Desktop/chronos-dist
```

这会将整个 dist 文件夹下载到桌面，命名为 `chronos-dist`。
